This fixture uses pnpm 7 and [`lockfileVersion` 5.4](https://github.com/pnpm/spec/blob/master/lockfile/5.2.md). 5.4 is almost identical for our purposes to 5.2, which was [introduced in pnpm 5](https://github.com/pnpm/spec/blob/master/lockfile/README.md).

Dependencies should match the other `basic-pnpm*` fixtures and should include at least one package with a peer dependency.
